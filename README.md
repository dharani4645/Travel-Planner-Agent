# Travel Planner Agent

A Jupyter Notebook-based project to intelligently plan trips and travel itineraries.

## Features

- Suggests trip itineraries based on destination, duration, and preferences
- Integrates with travel APIs to fetch flights, hotels, and attractions
- Customizable planning logic using Python and Jupyter Notebook
- Visualizes itineraries and routes in interactive cells

## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/dharani4645/Travel-Planner-Agent.git
    ```

2. **Install dependencies**  
   You may use a virtual environment.
    ```bash
    pip install -r requirements.txt
    ```

3. **Launch the notebook**
    ```bash
    jupyter notebook
    ```

4. **Open `TravelPlanner.ipynb`**  
   Follow the instructions in the cells to customize your trip planning.

## Usage

- Specify your travel dates, destination, budget, and other preferences in the notebook.
- Run the cells to generate and visualize your trip itinerary.

## Requirements

- Python 3.x
- Jupyter Notebook
- Additional dependencies as listed in `requirements.txt`
